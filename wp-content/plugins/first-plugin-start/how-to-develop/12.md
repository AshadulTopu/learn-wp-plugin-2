# Modify custom post type admin columns

### manage_post_type_posts_columns : This hook is used to modify the columns of the custom post type.

    --> add_filter('manage_my_post_type_posts_columns', 'my_custom_post_type_columns');

### reference : https://developer.wordpress.org/reference/hooks/manage_post_type_posts_columns/

manage and columns part are mandatory to modify custom post type columns (manage_post_type_posts_columns)

```php
add_filter('manage_my_custom_post_type_posts_columns', 'my_custom_post_type_columns');
function my_custom_post_type_columns($columns) {
    $columns['title'] = __('Title');
    $columns['date'] = __('Date');
    $columns['author'] = __('Author');
    $columns['categories'] = __('Categories');
    $columns['tags'] = __('Tags');
    return $columns;
}
```

why we need manage_post_type_posts_columns

--> this hook is used to modify the columns of the custom post type. like: title, date, author, categories, tags, product amount, product price, etc

# add custom column to custom post type

## https://developer.wordpress.org/reference/hooks/manage_post-post_type_posts_custom_column/

```php
add_action('manage_post-post_type_posts_custom_column', 'my_custom_column', 10, 2);
function my_custom_column($column, $post_id) {
    switch ($column) {
        case 'product_amount':
            echo get_post_meta($post_id, 'product_amount', true);
            break;
        case 'product_price':
            echo get_post_meta($post_id, 'product_price', true);
            break;
    }
}
```

```php
// let's say we have a CPT called 'product'
function product_custom_column_values( $column, $post_id ) {

    switch ( $column ) {

		// in this example, a Product has custom fields called 'product_number' and 'product_name'
        case 'product_number'	:
        case 'product_name' 	:
            echo get_post_meta( $post_id , $column , true );
        break;

		// in this example, $buyer_id is post ID of another CPT called "buyer"
        case 'product_buyer' 	:
            $buyer_id = get_post_meta( $post_id , $column , true );
            if( $buyer_id ){
                echo get_post_meta( $buyer_id , 'buyer_name' , true );
            } else {
                echo '<div class="dashicons dashicons-minus"></div>';
            }
        break;

    }
}
add_action( 'manage_product_posts_custom_column' , 'product_custom_column_values', 10, 2 );
```
