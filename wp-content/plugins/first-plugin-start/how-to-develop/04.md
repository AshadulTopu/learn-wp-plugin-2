# menu content into separate file

for separating file we need to know 2 url functions, and 4 file includes method

# url functions

1. plugin_dir_url( string $file ): string --> return url, example: http://localhost/first-plugin, use for css, js, image
2. plugin_dir_path( string $file ): string --> return path, example: /var/www/html/first-plugin, use for php file

then we need declare constant for these 2 functions

```php
define( 'FIRST_PLUGIN_URL', plugin_dir_url( __FILE__ ) );
define( 'FIRST_PLUGIN_PATH', plugin_dir_path( __FILE__ ) );
```

then we can use these constant in our plugin file

```php
wp_enqueue_style( 'first-plugin-style', FIRST_PLUGIN_URL . 'style.css' );
wp_enqueue_script( 'first-plugin-script', FIRST_PLUGIN_URL . 'script.js' );
```

```php
require_once( FIRST_PLUGIN_PATH . 'file.php' );
```

## reference : https://developer.wordpress.org/reference/functions/plugin_dir_url/

## reference : https://developer.wordpress.org/reference/functions/plugin_dir_path/

# include file method

1. require_once : --> require file only once, if file already included, it will not include again, PHP throws a warning but the script continues running
2. require : --> require file, if file already included, it will include again, PHP throws a warning but the script continues running
3. include_once : --> include file only once, if file already included, it will not include again, PHP throws a fatal error and the script stops.
4. include : --> include file, if file already included, it will include again, PHP throws a fatal error and the script stops, so it's not recommended
