# menu and submenu

### how to create menu

    --> add_menu_page('page_title', 'menu_title', 'capability', 'menu_slug', 'function', 'icon_url', 'position');

```php
    add_menu_page(
        'Page Title',
        'Menu Title',
        'capability',
        'menu-slug',
        'function',
        'dashicons-admin-generic',
        10
    )
```

### how to create submenu

    --> add_submenu_page('parent_slug', 'page_title', 'menu_title', 'capability', 'menu_slug', 'function');

```php
    add_submenu_page(
        'parent-slug',
        'Page Title',
        'Menu Title',
        'capability',
        'menu-slug',
        'function'
    )
```

### submenu as a parent menu

    --> add_submenu_page( 'parent_slug', 'page_title', 'menu_title', 'capability', 'parent_slug', 'function', 'icon_url', 'position');

```php
    add_submenu_page(
        'parent-slug',
        'Page Title',
        'Menu Title',
        'capability',
        'parent-slug',
        'function',
        'dashicons-admin-generic',
        10
    )
```

### submenu under submenu

    --> add_submenu_page('parent_slug', 'page_title', 'menu_title', 'capability', 'menu_slug', 'function');

```php
    add_submenu_page(
        'parent-slug',
        'Page Title',
        'Menu Title',
        'capability',
        'menu-slug',
        'function'
    )
```

### how to remove menu

    --> remove_menu_page('menu_slug');

```php
    remove_menu_page('menu-slug');
```

### how to remove submenu

    --> remove_submenu_page('parent_slug', 'submenu_slug');

```php
    remove_submenu_page('parent-slug', 'submenu-slug');
```

### how to remove menu and submenu

    --> remove_menu_page('menu_slug');
    --> remove_submenu_page('parent_slug', 'submenu_slug');

```php
    remove_menu_page('menu-slug');
    remove_submenu_page('parent-slug', 'submenu-slug');
```
