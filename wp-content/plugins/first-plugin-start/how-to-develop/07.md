# submenu on admin_bar_menu

### admin_bar_menu : This hook is used to add the admin bar menu.

    --> add_action('admin_bar_menu', 'my_admin_bar_menu', 999);

### reference : https://developer.wordpress.org/reference/hooks/admin_bar_menu/

```php
function my_admin_bar_menu() {
    global $wp_admin_bar;
    $wp_admin_bar->add_menu(array(
        'id' => 'my_admin_bar_menu',
        'title' => 'My Admin Bar Menu',
        'href' => '#',
    ));
}
add_action('admin_bar_menu', 'my_admin_bar_menu', 999);
```

# submenu on admin_bar_menu

```php
function my_admin_bar_menu() {
    global $wp_admin_bar;
    // main menu
    $wp_admin_bar->add_menu(array(
        'id' => 'my_admin_bar_menu',
        'title' => 'My Admin Bar Menu',
        'href' => '#',
    ));

    // submenu
    $wp_admin_bar->add_menu(array(
        'parent' => 'my_admin_bar_menu',
        'id' => 'my_admin_bar_submenu',
        'title' => 'My Admin Bar Submenu',
        'href' => '#',
    ))
}
add_action('admin_bar_menu', 'my_admin_bar_menu', 999);
```
