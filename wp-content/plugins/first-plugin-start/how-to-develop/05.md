# enqueue scripts and styles for the plugin

for enqueueing style and script we need to use wp_enqueue_style and wp_enqueue_script

```php
wp_enqueue_style( 'first-plugin-style', FIRST_PLUGIN_URL . 'style.css' );
wp_enqueue_script( 'first-plugin-script', FIRST_PLUGIN_URL . 'script.js' );
```

## function for enqueueing style and script

```php
function first_plugin_start_enqueue_scripts() {
    wp_enqueue_style( 'first-plugin-style', FIRST_PLUGIN_URL . 'style.css' );
    wp_enqueue_script( 'first-plugin-script', FIRST_PLUGIN_URL . 'script.js' );
}

// add_action('wp_enqueue_scripts', 'first_plugin_start_enqueue_scripts'); // Enqueue admin scripts and styles for frontend only
// add_action('admin_enqueue_scripts', 'first_plugin_start_enqueue_scripts'); // Enqueue admin scripts and styles for admin area only
add_action('init', 'first_plugin_start_enqueue_scripts');   // Enqueue admin scripts and styles for frontend and admin both areas
```

### Note: for enqueueing style and script for plugin / admin page we need to use add_action hook with "init" hook name

```php
add_action( 'init', 'my_basics_plugin_enqueue_scripts' );
```
