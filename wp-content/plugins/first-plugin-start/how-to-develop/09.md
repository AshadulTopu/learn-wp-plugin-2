# add metabox

### add_meta_box : This hook is used to add the metabox.

    --> add_action('add_meta_boxes', 'my_add_meta_box');

### reference : https://developer.wordpress.org/reference/hooks/add_meta_boxes/

```php
function my_add_meta_box() {
    add_meta_box(
        'my_meta_box',
        'My Meta Box',
        'my_meta_box_callback',
        'post',
        'normal',
        'high'
    );
}
add_action('add_meta_boxes', 'my_add_meta_box');

function my_meta_box_callback($post) {
    echo '<input type="text" name="my_meta_box" value="' . get_post_meta($post->ID, 'my_meta_box', true) . '">';
}
```

## save metabox

### save_post : This hook is used to save the metabox.

    --> add_action('save_post', 'my_save_meta_box');

### reference : https://developer.wordpress.org/reference/hooks/save_post/

```php
function my_save_meta_box($post_id) {
    if (isset($_POST['my_meta_box'])) {
        update_post_meta($post_id, 'my_meta_box', sanitize_text_field($_POST['my_meta_box']));
    }
}
add_action('save_post', 'my_save_meta_box');
```
