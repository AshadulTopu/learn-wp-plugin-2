# hook

### what is hook

    --> function called when something happens in wordpress.

### type of hook

    --> action hook
    --> filter hook

### how to create hook

    --> function name must start with "add_"

### how to add hook

    --> add_action('hook_name', 'function_name');
    --> add_filter('hook_name', 'function_name');

### how to remove hook

    --> remove_action('hook_name', 'function_name');
    --> remove_filter('hook_name', 'function_name');

### how to use hook

    --> do_action('hook_name');
    --> apply_filters('hook_name', 'value');

### hook priority

    --> higher number = higher priority

```php
<?php
add_action( string $hook_name, callable $callback, int $priority = 10, int $accepted_args = 1 ): true
```

### reference : https://developer.wordpress.org/reference/functions/add_action/

## filter

```php
<?php
add_filter( string $hook_name, callable $callback, int $priority = 10, int $accepted_args = 1 ): true
```

### reference : https://developer.wordpress.org/reference/functions/add_filter/
