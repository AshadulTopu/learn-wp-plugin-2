# notices

### notices : This hook is used to add notices.

    --> add_action('admin_notices', 'my_admin_notices');

### reference : https://developer.wordpress.org/reference/hooks/admin_notices/

```php
function my_admin_notices() {
    echo '<div class="notice notice-success"><p>My Notice</p></div>';
}
add_action('admin_notices', 'my_admin_notices');
```

### error : This hook is used to add error notices.

```php
function my_admin_notices() {
    echo '<div class="notice notice-error"><p>My Notice</p></div>';
}
add_action('admin_notices', 'my_admin_notices');
```

### dismissable : This hook is used to add dismissable notices.

```php
function my_admin_notices() {
    echo '<div class="notice notice-success is-dismissible"><p>My Notice</p></div>';
}
add_action('admin_notices', 'my_admin_notices');
```
