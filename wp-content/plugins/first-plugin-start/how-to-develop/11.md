# custom post type

### register_post_type : This hook is used to register the custom post type.

    --> add_action('init', 'my_register_post_type');

### reference : https://developer.wordpress.org/reference/hooks/init/

```php
function my_register_post_type() {
    register_post_type('my_custom_post_type', array(
        'labels' => array(
            'name' => 'My Custom Post Type',
            'singular_name' => 'My Custom Post',
        ),
        'public' => true,
        'has_archive' => true,
    ));
}
add_action('init', 'my_register_post_type');
```
